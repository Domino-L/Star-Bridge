# 星海舰桥 Star Bridge

星海舰桥是一个面向《星际公民》的舰队辅助工具。

制作本应用的目标为补齐游戏内暂时缺少的舰队管理能力：舰队、成员、小队、任务、集结点、Overlay 状态显示，以及基于 `Game.log` 的本地状态辅助识别。

当前测试版本：`0.3.11`

## 现阶段定位

星海舰桥可以帮助舰队更清晰地组织成员，让玩家更深度地参与到舰队行动中。

目前重点能力：

- 创建或加入舰队
- 创建、加入和同步小队
- 查看舰队成员、小队成员、在线状态、飞船状态、地点信息
- 发布舰队任务、集结点和行动计划
- 通过 Overlay 在游戏画面上方显示舰队与小队信息
- 读取 Star Citizen `Game.log`，辅助判断玩家身份、飞船和位置
- 通过中继服务器同步舰队、小队与基础玩家状态
- 使用邮箱验证码注册账号，并以登录邮箱绑定个人呼号

## 合规原则

星海舰桥只读取日志、展示信息、同步用户主动上报或日志推断出的状态。

它不会：

- 注入游戏
- Hook 渲染或输入
- 读取游戏内存
- 自动操作游戏
- 修改游戏文件

## 用户安装包

当前用户安装包：

```text
dist/StarBridge-0.3.11-win-x64-setup.exe
```

这是完整 Windows 安装包，不要求用户额外安装 .NET Desktop Runtime。

## 中继服务器

当前中继服务器默认地址：

```text
https://api.scstarbridge.com
```

客户端登录后会自动连接服务器并启用同步能力。

## 使用流程

1. 安装并打开星海舰桥。
2. 首次进入时登录或注册账号；也可以跳过，以浏览模式进入。
3. 选择 Star Citizen 的 `Game.log`。
4. 进入游戏，让软件从日志中识别玩家游戏名。
5. 创建舰队，或在“寻找舰队”中加入现有舰队。
6. 创建或加入小队。
7. 在 “Overlay” 页面调整显示布局与预设。
8. 打开 Overlay，在游戏中查看舰队任务、小队和成员状态。

## 浏览模式与登录模式

未登录时可以：

- 浏览界面
- 查看基础页面结构
- 调整部分 Overlay 布局

未登录时不能：

- 同步玩家状态
- 创建或加入舰队
- 创建或加入小队
- 发布任务、公告和计划
- 使用舰船数据库

登录后的意义：

- 账号通过登录邮箱识别
- 注册时填写的呼号会绑定到个人身份
- 舰队与小队管理行为会和账号关联
- 可使用联网同步、舰船数据库和舰队管理功能

## 开发与打包

从源码运行桌面应用：

```powershell
.\scripts\Start Fleet Command Designer.cmd
```

构建完整安装包：

```text
scripts/Build Star Bridge Inno Installer.cmd
```

生成结果：

```text
dist/StarBridge-0.3.11-win-x64-setup.exe
```

构建完整安装包需要：

- .NET 8 SDK
- Inno Setup 6
- 首次 self-contained 发布时需要联网下载 .NET runtime packs

如果打包时报 `NU1100`，说明本机缺少 .NET 运行时包缓存，请在联网状态下重新运行打包脚本。
